fun.pF2h(soilwaterfun)R Documentation

Converts pF to h

Description

Function that converts pF values into a matrix poential (in meters [m]).

Usage

fun.pF2h(pF)

Arguments

pF Vector of numericals. pF values to be converted.

Value

Returns a vector of matrix potential value [m] (numericals), negative (succion).

fun.pF2h( pF = c(0,2,4.2) )
