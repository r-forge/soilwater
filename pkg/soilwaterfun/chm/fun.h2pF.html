fun.h2pF(soilwaterfun)R Documentation

Converts h to pF

Description

Function that converts matrix potential values [m] into pF values

Usage

fun.h2pF(h)

Arguments

h Vector of numericals. Matrix potential values to be converted [m] (negative values. Succion).

Value

Returns a vector of pF values (numericals)

Author(s)

Julien MOEYS <jules_m78-soiltexture@yahoo.fr>

Examples

require( "soilwaterfun" )

fun.h2pF( h = fun.pF2h( pF = c(0,2,4.2) ) )
